# Data Science Internship Projects

## Overview

This repository contains three Data Science internship projects completed as part of my internship program. These projects demonstrate skills in data analysis, data preprocessing, visualization, exploratory data analysis (EDA), machine learning, and business insight generation.

## Projects

### 1. Loan Approval Prediction

* Performed data cleaning and preprocessing.
* Built machine learning models to predict loan approval status.
* Evaluated model performance using appropriate metrics.

### 2. Zomato Data Analysis

* Analyzed restaurant data to identify customer preferences and business trends.
* Performed exploratory data analysis and visualization.
* Generated actionable business insights.

### 3. Instagram Engagement Analysis

* Analyzed Instagram comment and engagement data.
* Studied user activity patterns, hashtags, emojis, and sentiment.
* Developed content strategy recommendations to improve engagement.
